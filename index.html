What is CSS?

  1. CSS stands for cascading style sheet

  2. CSS describes how HTML, Elements are to be displayed on screen ,paper,or in other media.

  3. CSS is not necessary. CSS Saves alot of work. It can control the layout of multiple web pages all at once.

  4. External stylesheets are stored in CSS files


    5. CSS can be added to HTML documents in 3 ways


    • Inline- By using the style attribute inside HTML elements
    • Internal- By using a style element in the head section
    • External- By using a link element to link to an external CSS file
    • The most common way to add CSS is to keep the styles in external CSS file.However in this tutorial we will use inline and internal styles, Because this is easier to demonstrate and easier for you to do it yourself.
